Схема URL

Уніфіко́ваний лока́тор ресу́рсів або адре́са ресу́рсу (англ. Uniform Resource Locator — єдиний вказівник на ресурс, URL) — стандартизована адреса певного ресурсу (як-от документ або зображення) в інтернеті

URL складається з декількох частин, які визначають місце розташування ресурсу:

  • Протокол: Це перший шматок в URL і вказує, який протокол зв'язку використовується для доступу до ресурсу. Найпоширенішими протоколами є HTTP (Hypertext Transfer Protocol) і HTTPS (HTTP Secure) для веб-сайтів, FTP (File Transfer Protocol) для передачі файлів, і так далі.
  • Домен: Це ім'я або IP-адреса сервера, на якому знаходиться ресурс. Наприклад, у URL "https://www.example.com", "www.example.com" - це домен.
  • Шлях до ресурсу: Ця частина вказує на конкретний ресурс або сторінку на веб-сайті. Наприклад, у URL "https://www.example.com/page1/page2", "/page1/page2" - це шлях до ресурсу.
  • Параметри запиту (опціонально): Параметри запиту додаються після знака питання (?) і використовуються для передачі додаткової інформації на сервер. Наприклад, у URL "https://www.example.com/search?q=url", "?q=url" - це параметр запиту, який передає запит на пошук.
